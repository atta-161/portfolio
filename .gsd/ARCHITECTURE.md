# Architecture

> Generated by /map on 2026-05-16

## Overview
This is a single-page personal portfolio built with Next.js App Router. The home route composes six client-rendered sections into a dark, animated portfolio experience inspired by Rauno Freiberg: Hero, About, Projects, Skills, Experience, and Contact.

The app uses static local data for portfolio content, local project images, Framer Motion for interaction/scroll animations, and Web3Forms for client-side contact submission.

## Data Flow
- `src/data/projects.ts` exports `PROJECTS`; `ProjectGallery` filters the array and renders `ProjectCard`.
- `src/data/skills.ts` exports `SKILL_CATEGORIES`; `SkillsShowcase` selects key skills for a bento layout and passes them to `SkillCard`.
- `src/data/experience.ts` exports `EXPERIENCES`; `ExperienceTimeline` maps entries into timeline rows.
- `ContactSection` collects form fields, appends `NEXT_PUBLIC_WEB3FORMS_KEY`, and posts JSON to Web3Forms.
- `Navbar` uses `IntersectionObserver` to update active state and custom `requestAnimationFrame` easing for click navigation.

## Conventions
- **Components:** PascalCase React components in `src/components` and `src/components/sections`.
- **Data:** Typed static arrays in `src/data`.
- **Styling:** Tailwind utility classes plus small global utilities (`glass`, `text-muted`, `glow-hover`).
- **Animations:** Framer Motion variants, `whileInView`, hover transforms, layout animations, and custom scroll easing.
- **Imports:** `@/*` path alias maps to `src/*`.
