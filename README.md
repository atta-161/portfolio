# portfolio

## Introduction

This is a portfolio website for me. It is a web application that is built with Next.js and Tailwind CSS. Sources code from antigravity (Claude Opus 4.5, Gemini Flash, GPT-5.5). 

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
```

```bash
npm run start
```

## Linting

```bash
npm run lint
```
